# Clawdbot Explorer

An interactive, visually-rich educational interface that explains Clawdbot in an intuitive, self-evident way. Designed to be exploratory and visually engaging with animations and interactive elements.

## Features

- Interactive visualization of Clawdbot architecture
- Animated transitions and micro-interactions
- Explorable sections covering all aspects of Clawdbot
- Visually appealing design with modern UI patterns
- Responsive layout for all device sizes

## Tech Stack

- Next.js 14 with App Router
- React with TypeScript
- Framer Motion for animations
- CSS Modules for styling
- Local storage for persistence

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Design Philosophy

This interface is designed to be:
- Self-evident: Users understand the system at a glance
- Explorable: Interactive elements encourage discovery
- Visually rich: Animations and graphics enhance understanding
- Educational: Clear explanations of complex concepts
- Intuitive: Natural navigation and interaction patterns

## Sections

- Overview: High-level introduction to Clawdbot
- Architecture: Detailed system architecture visualization
- Features: Interactive demonstration of capabilities
- Demo: Simulated environment to try features
- Integration: Options for connecting platforms and services
